What is StacTab

Tab chaos is a solved problem.

StacTab is a lightweight Chrome extension that brings order to your browser automatically. It groups related tabs by domain, sleeps memory-hungry background tabs, zaps duplicates, and gives you a searchable archive for every session you've ever worked on.

Built for developers, researchers, and anyone who's ever had 40+ tabs open and lost track of what they were even doing. Free, open source, and respects your privacy — no data ever leaves your browser.

Features

Everything you need.
Nothing you don't.

Every feature is designed to reduce friction, not add it. One click is always enough.

⚡
Auto-Pilot Mode
Set it and forget it. StacTab watches your tabs in real time and automatically groups tabs from the same domain and applies color-coding the moment your configurable threshold (2–5 tabs) is crossed.
Automatic
🗂️
Smart Organize
One click groups all open tabs by domain into named, color-coded Chrome tab groups. Perfect when you've been browsing freely and want to instantly bring order to chaos.
One Click
💤
Sleep Tabs
Unloads inactive background tabs from RAM without closing them. Your tabs stay in the tab bar — they just stop consuming memory until you click on them again. Reclaim hundreds of megabytes instantly.
Memory Saver
🔁
Dedupe Tabs
Scans all open tabs for identical URLs and closes every duplicate in a single action, keeping only one copy of each. Great for when you've accidentally opened the same page a dozen times.
Instant Clean
🪟
Merge Windows
Pulls all tabs from every other Chrome window into the current one. Stop context-switching between windows — consolidate everything into one workspace with a single click.
Window Control
🔍
Live Tab Search
Instantly filter all open tabs by title or URL as you type. Hit AltS from anywhere to open StacTab and jump directly to the search. No more hunting through tab rows.
Alt+S
🔢
Tab Count Badge
Always know exactly how many tabs you have open. StacTab displays a live count badge directly on the extension icon in your toolbar — always visible, always accurate.
Always Visible
🗄️
Workspace Vault
Archive entire sessions, individual tabs, or tab groups into a searchable, sortable vault. Name your sessions, restore them later, export to JSON or .txt, and undo any archive within 30 seconds.
⭐ Flagship

Workspace Vault

Your sessions.
Safe, searchable, forever.

The Workspace Vault is StacTab's most powerful feature — a fully-featured tab archive that lives entirely in your browser.

  • 📛
    Named Sessions Save groups of tabs as "Work", "Research", "Shopping" — whatever makes sense to you.
  • 📦
    Flexible Archive Options Archive the current tab, all tabs, tabs to the left/right, or an entire tab group in one action.
  • 🪟
    Split Groups to Windows Explode any tab group into its own new Chrome window — perfect for deep focus sessions.
  • ♻️
    Smart Restore Restore individual tabs, all tabs from a domain, all tabs from a day, or an entire session in a new window.
  • ↩️
    30-Second Undo Accidentally archived the wrong thing? A countdown undo button lets you instantly reverse it.
  • 📤
    JSON & TXT Export Export your entire vault as structured JSON for backup or portability, or as a plain .txt reading list.

Get Started

Up and running
in under 60 seconds.

No Chrome Web Store listing (yet). Install directly from source — it takes less than a minute.

1
Download the extension

Go to the GitHub repo and click Code → Download ZIP and unzip it — or clone it with the command below.

2
Open Chrome Extensions

In a new Chrome tab, navigate to chrome://extensions/

3
Enable Developer Mode

Toggle the Developer mode switch in the top-right corner of the Extensions page to ON.

4
Load Unpacked

Click the Load unpacked button and select the unzipped StacTab folder.

5
You're done!

StacTab appears in your toolbar. Pin it for quick access and press AltS anywhere to open it instantly.

bash
# Clone the repository git clone https://github.com/MananSharma26/StacTab.git # Or download the ZIP from GitHub # Code → Download ZIP → Unzip # Then in Chrome: # chrome://extensions/ → Developer mode ON # → Load unpacked → select /StacTab
💡
Pro tip: After installing, press AltS from any tab to instantly open StacTab and search across all your open tabs — no need to click the toolbar icon.
🔒
Fully private. StacTab only uses the tabs, tabGroups, storage, and contextMenus permissions. No network requests. No analytics. No tracking. Ever.
