AUTO-PILOT

          
One click. Every tab finds its home.

          
Set a threshold — 2, 3, or 4 tabs from the same domain — and StacTab groups them automatically. No clicks, no drag-and-drop, no menu diving.

          
Alt+G

        

      


      

        

          
SLEEP + DEDUPE

          
Your RAM didn't sign up for this.

          
StacTab sleeps tabs that have been idle. Duplicates are closed automatically. The tab stays visible — the memory footprint doesn't.

        

      


      

        

          
LIVE SEARCH

          
Cmd+F, but for every tab you have open.

          
Search all open tabs by title and URL. Hit Alt+S anywhere — you don't even need to open the popup. Results appear instantly.

          
Alt+S · Option+S on Mac
